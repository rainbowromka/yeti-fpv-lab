# Yeti FPV Lab

Лаборатория внутренних систем FPV-дрона для съёмки и автопилотов сноубордистов и лыжников в горах.

## Возможности

- **3D-сцена**: горный склон 1400×200 метров, коридор с лесным массивом
- **Сноубордист**: движется по трассе со случайной извилистостью
- **Дрон-камера**: реалистичная физика полёта (гравитация, тяга, сопротивление воздуха)
- **Управление стиками**: плавный набор газа, рыскание, тангаж, крен
- **Автопилот**: ПИД-регуляторы для удержания высоты и следования за райдером
- **Телеметрия**: отображение параметров полёта в реальном времени

## Управление

| Клавиша | Действие |
|---------|----------|
| W/S | Газ вверх/вниз |
| A/D | Рыскание влево/вправо |
| I/K | Тангаж вперёд/назад |
| J/L | Крен влево/вправо |
| Q | Сброс газа и углов |
| F | Автопилот (следование за райдером) |
| Пробел | Пауза сноубордиста |
| R | Перезапуск сноубордиста |
| T | Новая трасса |
| ESC | Выход |

## Установка и запуск

```bash
python -m venv venv
source venv/bin/activate
pip install PyQt5 PyOpenGL PyOpenGL_accelerate numpy
python main.py
```

## Структура проекта

```
yeti-fpv-lab/
├── main.py              # Точка входа, главное окно
├── gl_widget.py          # OpenGL виджет
├── scene.py              # 3D сцена
├── info_panel.py         # Панель телеметрии
├── snowboard.py          # Сноубордист и траектория
├── corridor.py           # Коридор и лес
├── slope.py              # Склон
├── grid_plane.py         # Координатные плоскости
├── camera/               # Пакет камеры
│   ├── camera.py         # Дрон-камера
│   ├── stick.py          # Плавное управление стиками
│   └── pid.py            # ПИД-регулятор
└── run.sh                # Скрипт запуска
```
